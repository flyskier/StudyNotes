**作用**

* 生成目录结构

* 本地开发调试

* 代码部署

* 热加载

* 单元测试

**安装**
      
      window+R输入 cmd进入命令行
      
      全局安装
      npm install -g vue-cli
     
     查看版本号
      vue -V
      vue -Version
      
      进入项目存储盘如d盘ProjectTest
       进入d盘
