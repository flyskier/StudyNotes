JavaScript模块化开发
==========

**目的**

1.为了解决全局的方法和变量带来的冲突  

      var a=10;
      function tab(){}

1.使用[requirejs](http://requirejs.org/docs/start.html])

2.使用[seajs](https://seajs.github.io/seajs/docs/#docs)（作者已经不维护了）

3.使用ES6的原生模块化方案（主流）
